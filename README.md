# Strayer's Skyblock Utils
## What is it?

Strayer's Skyblock Utils is my way of contributing to the Hypixel Skyblock modding community. It contains a few tweaks that, in my opinion, make the gameplay smoother and more enjoyable.

SSU started as a personal project. I wanted to make something for myself to help me in some aspect of the games I wish other mods tweaked, or I didn't like the way other mods did.

<details>
<summary>Features:</summary>

### Dungeons
- Dungeon join commands in "!M7" format
- Floor auto-rejoin via /ssu autorejoin (floor("m7" format))
- Downtime tracker via !dt/dt (reason(optional))
### Mining
- Corleone reminders via keybinds (reminds you 1 minute after killing it)
- Umber and tungsten veins waypoints while in the glacite tunnels
### Chat
- Chat filter toggles (Incomplete)
- Printing coordinates to chat via keybinds
- Fancy emotes (o/ into ( ﾟ◡ﾟ)/ and such)
### Glowing players
- Glowing players via /ssu glowingPlayers
- Config hud for editting glowing players
- Automatic party members glow
### Misc
- A customizable contextual hud
- Custom party invite notifications
- Auto Hoppity eggs
- Zoom
- Chat peek
- Color styles
- Custom sidebar
</details>

## Dependencies
- Cloth Config Api - Mandatory
- Mod Menu - Recommended
