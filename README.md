# Strayer's Skyblock Utils
Adds a few useful utils to Skyblock  
<details>
  <summary>Feature list:</summary>

  - Corleone Timer
  - Chat Filters
  - Fancy Emotes
  - Glowing Players
  - Party Invite Notifications
  - Contextual HUD
  - Dungeon Floor Commands
  - Downtime Tracker
  - A few somewhat useful commands (/ssu)
  - Umber and tungsten waypoints for glacite tunnels
  - Party commands (!pt !w and such)
  - !ping !tps and !fps in any chat
  - Party members tracker
</details>
